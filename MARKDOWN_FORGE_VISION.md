# Markdown Forge

Mission

Help people build knowledge that maximizes an AI's ability to understand, reason, remember, and act.

---

North Star

Does this Markdown enable an AI to do something it otherwise couldn't?

---

Principles

- Free version must provide real value.
- AI should enhance the Forge Engine, not replace it.
- Every feature should improve AI-ready knowledge.
- Build products first, learn technology second.
